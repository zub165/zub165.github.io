How to Delete Your Data in AI Agent

Your privacy is important to us. If you'd like to delete your data, please follow the steps below:

🔄 Using the "Clear Chat" Feature

You can clear your chat history by tapping the “Clear Chat” button inside the AI Agent app. This permanently deletes your current conversation history from the backend and cannot be recovered.

📧 Request Full Data Deletion

If you’ve integrated a custom server or would like to request any server-side data removal, please contact us:

  • Email: zub165@yahoo.com
  • GitHub: Submit an issue

🔒 Data Retention

AI Agent does not store personal health data unless you’re using a custom backend. By default, no identifiable information is stored beyond the session.
